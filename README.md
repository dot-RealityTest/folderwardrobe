# folderwardrobe

`folderwardrobe` is a macOS app for customizing Finder folders with color, custom icons, and reusable metadata presets.

## Download

Download the latest notarized DMG:

- [folderwardrobe.dmg](https://github.com/dot-RealityTest/folderwardrobe/releases/latest/download/folderwardrobe.dmg)

Website:

- [akakika.com/folderwardrobe](https://akakika.com/folderwardrobe/)

## What You Can Do

- Change folder color and tint
- Replace the default folder icon with a custom one
- Add Finder comments and tags
- Save reusable icon and metadata presets
- Preview changes before applying them
- Revert a folder back to its previous look

## How It Works

1. Add folders by drag-and-drop or with the folder picker.
2. Select a folder from the list.
3. Adjust color, icon, Finder comment, or tags.
4. Preview the result.
5. Apply the changes or save the setup as a reusable preset.

## Requirements

- macOS 13 or later

## Notes

- Finder metadata can take a moment to refresh visually.
- Some copy or move workflows may remove Finder metadata.
- Protected or system folders may fail due to macOS permissions.
