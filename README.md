## 🎨 Icon Collection Details

### Design Philosophy

The icon collection was built with three principles:

1. **Apple HIG Compliance** — Every icon follows Human Interface Guidelines for spacing, padding, and visual weight
2. **Manual Craftsmanship** — Hand-cropped and refined, not AI-generated
3. **macOS Integration** — Designed to look native, not out of place

### Icon Categories

- **Work:** Briefcase, documents, charts, presentations
- **Creative:** Camera, palette, music, video
- **Personal:** Home, heart, travel, health
- **System:** Settings, terminal, database, network
- **Media:** Photos, music, videos, podcasts
- **Education:** Books, courses, notes, research

### Technical Specs

- **Format:** SVG + PNG (Retina optimized)
- **Sizes:** 16px, 32px, 64px, 128px, 256px, 512px, 1024px
- **Color Space:** sRGB with P3 wide gamut support
- **Export:** Multiple resolutions for all macOS versions

## 🙋 FAQ

**Q: Does this work on all macOS versions?**  
A: folderwardrobe supports macOS 12.0 (Monterey) and later.

**Q: Can I use my own icons?**  
A: Yes! Upload any SVG or PNG file.

**Q: Are the presets synced across devices?**  
A: Currently local only. iCloud sync planned for future version.

**Q: Can I revert changes?**  
A: Yes! Right-click any folder in Finder → "Get Info" → click the icon → press Delete to revert to default.

**Q: Is it free?**  
A: Yes, completely free.
